Data Science Training

Data science is a term that covers many scopes, from careers to work practices, it touch every facets of modern life.
Recording data is as old as counting, but what is new is the way data science use special tools and techniques to analyze data for meaningful insights that greatly benefit users.

Python has in recent years become and remained the default language with first-class tool of libraries used for scientific computing tasks such as data analysis, data visualization and Machine Learning (ML) of large datasets, of which common and conventional non-programming data analysis packages like excel, STATA, SPSS, MATLAB etc will not dream of handling

The advantages of using Python tools for data science stems primarily from the large and active ecosystem of third-party complimenting packages like:

  • NumPy and Pandas for manipulating homogeneous array data and heterogenous labeled data respectively
  • Anaconda full stack package used by professional data scientists
  • SciPy for scientific computing tasks
  • Matplotlib for publication of quality data visualizations
  • IPython for interactive execution and sharing of code
  • Scikit-Learn for Machine Learning
  • ...and many more tools that will be taught as you enroll.
