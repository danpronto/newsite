Data Science Course Overview

Certification: Applied Certificate in Data Science with Python

Duration: Self-paced (56 hrs)

Location: Abuja

Cost: N120,000

In this Data science course, you will learn from the basics of conducting data science project, how to perform data analysis in python, how to apply the technicalities and knowledge acquired to solving real life problems and facilitating informed decision making.

This course will introduce you to the field of data science and will prepare you for other specialized data science areas like Machine Learning (ML), Apache Spark, Hadoop, robotics and artificial intelligence (AI), Big Data Analytics, Convolution Neural Networks (CNN) First, and foremost, you'll learn how to do data science by learning how to analyze data.

This include knowing how to import data, explore it, analyze it, learn from it, visualize it, and ultimately generate easily shareable reports. To really do data analysis with python, we will teach you a collection of powerful, open-source, tools including: Python, Jupyter notebooks, Pandas, NumPy, Matplotlib, Seaborn, and you won't be learning these tools in isolation in this course.
You will be learning these tools all within the context of compelling real life data sets.

Specific learning Objectives of the course include developing the basic process of doing data science, mastering how to use Jupyter notebooks (the main environment for doing data science), applied understanding manipulating NumPy arrays, data cleaning and preparation, being comfortable with visualizing data and results.
You will also learn how to use functions to process NumPy data in manners which are time and space efficient than Python data object.

We will specifically be covering broad and imperative topics such as but not limited to:

  • Understanding data types in Python versus NumPy
  • NumPy array attributes
  • Indexing in NumPy
  • Broadcasting in NumPy
  • Sorting arrays in NumPy
  • Pandas objects
  • Handling missing data in Pandas
  • Combining datasets in Pandas
  • Aggregation in Pandas
  • Vectorized string operations

When working with large datasets, you often need to visualize your data to gain a better understanding of it. Also, when you reach conclusions about the data, you'll often wish to use visualizations to present your results. This is why Matplotlib is a cannot-do-without Python for data science library.
